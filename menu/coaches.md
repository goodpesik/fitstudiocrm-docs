# 3.4 Тренери

![](../_media/coaches-1.png ':size=200x70')

## 3.4.1 Картка тренера

> Картка тренера містить:

- **Імʼя**
- **Аватар** (якщо завантажено; інакше використовується стандартний)
- **Телефон**
- **Кнопки**:
  - **Видалити** — неможливо видалити тренера, у якого є заплановані тренування
  - **Редагувати**

---

## 3.4.2 Додати тренера

> Кнопка **Додати тренера** знаходиться над списком карток та позначена іконкою «+».

> Після натискання відкривається модальне вікно:

![](../_media/coach-2.png ':size=150x110')

**Містить:**

- Завантаження аватара
- Текстове поле **Імʼя** (обовʼязкове)
- Поле **Телефон** (обовʼязкове, цифровий формат)
- Колір-пікер для вибору кольору відображення тренера у календарі (обовʼязкове поле)
- Вибір **типу ставки за замовчуванням** (доступно 2 типи) — застосовується до **всіх категорій тренера, для яких індивідуальну ставку не задано окремо**:
  - **За кількістю людей на тренуванні** (`PerPerson`) — зарплата = `кількість відвідувань × ставка категорії`.
  - **За кількістю тренувань** (`PerWorkout`) — зарплата = `кількість проведених тренувань × ставка категорії`.

  Детальніше: [Інформація про обробку візитів, списання абонементів та нагадування](/pages/processing)

![](../_media/coaches-3.png ':size=100x110')

- **Кнопка «Додати категорію»** — дозволяє додати одну з уже створених категорій та задати для тренера **індивідуальну ставку** саме для цієї категорії.

> ℹ️ **Як вирішується який тип ставки і яке значення застосувати при підрахунку зарплати:**
> 1. Якщо у тренера для категорії задано індивідуальну ставку → використовується вона (значення + тип з категорії).
> 2. Якщо ні — береться тип ставки за замовчуванням з картки тренера + базова ставка категорії зі сторінки [Звіти → Категорії](/menu/reports#_383-Категорії).
> 3. Тип ставки самої категорії (`PerPerson` / `PerWorkout`) на сторінці категорій визначає форму розрахунку, але кількісне значення береться з картки тренера або індивідуальної категорії-надбавки.
