# Телеграм-бот / Мобільний додаток

> ⚠️ **Сторінку залишено для зворотної сумісності посилань.**  
> Окремої вкладки **Телеграм-бот** в Адмін-панелі більше **немає** — функціонал клієнтського додатку (QR-код для встановлення, лічильники зареєстрованих клієнтів/тренерів, масова розсилка повідомлень) переїхав у вкладку **Мобільний додаток**.

➡️ Актуальна документація: [Налаштування → Мобільний додаток](/login/settings#мобільний-додаток).
